## falling_down
#### 产品简介：
> 《梦境坠落》是一款跑酷类型的休闲游戏。以主角进入梦乡做了一个不断坠落的梦为主题，玩家通过在不同梦境躲避不同障碍物进行生存跑酷游戏。主角在梦中活得越久，分数就越高。
> 整个游戏过程中有三个坠落位置，分别为左、中、右，切左右可以循环切换。在游戏中可以获得金币，钻石等道具。同时玩家的高分可以通过人人转发告知好友。
